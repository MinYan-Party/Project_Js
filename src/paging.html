效果示例

 

导入文件

<link rel="stylesheet" href="../css/paging.css" />
<script type="text/javascript" src="jquery1.11.1.min.js"></script>
<script type="text/javascript" src="../js/paging.js"></script>

调用示例

  $('.pageTurning').paging(
      {
        itemClick:function(page,obj){
          obj.setData({page:page,totalNum:110});  // totalNum:总页数
        }
      }
  );

              url:'',

                  /** 处理逻辑 开始 */

                  ...

                  /** 处理逻辑 结束 */
                  obj.setData({page:page,totalNum:obj.options.totalNum});   //page不用理会，totalNum为调用ajax分页后返回的列表总行数（需自行处理）

参数说明

名称 描述
pageSet 分页常用参数。
totalpage：总页数，在插件中计算，不用手动设置，默认就行；
page：当前页码，设置后该页码为选中状态。
totalNum 总记录数，插件调用时设置
num 每页显示行数
position 分页显示位置，可选参数（left、center、right），PS:可能存在样式冲突而不起作用的问题

方法说明

方法 描述
itemClick 点击分页页码时触发，具体使用参照“调用示例”
